# YourTrackOrderBot / TrackOrderPro

Telegram-бот для селлеров WB/Ozon: статусы заказов (пока заглушка) + шаблоны ответов и ночной автоответ.

## Что умеет MVP

- Команда `/start` — показывает меню и краткую инструкцию.
- `/track` — заглушка статуса заказа (позже будет реальный WB API).
- `/templates` — быстрые шаблоны ответов (инлайн-кнопки).
- `/night` — ночной режим с автоответом "статус утром".
- `/pay` — кнопка оплаты 300₽/мес через YooKassa.

## Как запустить локально

1. Установить Python 3.11+.
2. Клонировать репозиторий:
   ```bash
   git clone https://github.com/stasyk411/YourTrackOrderBot-.git
   cd YourTrackOrderBot-
   ```
